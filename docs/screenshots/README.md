# Скриншоты приложения

Эта папка содержит скриншоты для документации проекта.

## Необходимые скриншоты

Для полной документации нужно сделать следующие скриншоты:

### 1. homepage.png
- **Страница:** Главная страница (/)
- **Содержание:** Навигация, баннер, секции с рекомендациями

### 2. register.png
- **Страница:** Регистрация (/register)
- **Содержание:** Форма регистрации с полями для email, username, password, любимых жанров

### 3. catalog.png
- **Страница:** Каталог (/catalog)
- **Содержание:** Сетка книг, фильтры сбоку, пагинация

### 4. search.png
- **Страница:** Поиск в каталоге
- **Содержание:** Строка поиска, результаты поиска по запросу "фантастика"

### 5. book-detail.png
- **Страница:** Детали книги (/books/:id)
- **Содержание:** Информация о книге, кнопки действий, секция "Похожие книги"

### 6. recommendations.png
- **Страница:** Персональные рекомендации (главная страница, секция "Для вас")
- **Содержание:** Карточки рекомендованных книг на основе истории

### 7. trending.png
- **Страница:** Популярное (главная страница, секция "Популярное")
- **Содержание:** Трендовые книги

### 8. cart.png
- **Страница:** Корзина (/cart)
- **Содержание:** Список книг в корзине, итоговая сумма, кнопка оформления заказа

### 9. profile.png
- **Страница:** Профиль пользователя (/profile)
- **Содержание:** Личная информация, любимые жанры, история заказов

### 10. swagger.png
- **Страница:** API документация (http://localhost:8000/docs)
- **Содержание:** Swagger UI с endpoints

## Как делать скриншоты

### Для Windows:
1. Используйте `Win + Shift + S` для выделения области
2. Или используйте Snipping Tool / Snip & Sketch

### Для Mac:
1. `Cmd + Shift + 4` для выделения области
2. Или используйте встроенный Screenshot

### Для Linux:
1. Используйте `gnome-screenshot -a` для выделения
2. Или используйте Shutter, Flameshot

## Рекомендации

- **Разрешение:** Минимум 1920x1080
- **Формат:** PNG (для лучшего качества)
- **Размер:** Оптимизируйте до < 500KB если возможно
- **Содержимое:** Убедитесь, что нет личных данных
- **Полнота:** Покажите ключевые функции на скриншоте

## Альтернатива

Если вы не хотите делать скриншоты вручную, можете использовать placeholder изображения для документации:

```bash
# Создать placeholder изображения
for file in homepage register catalog search book-detail recommendations trending cart profile swagger; do
  # Используйте любой онлайн сервис для генерации placeholder
  # Например: https://placeholder.com/
done
```
